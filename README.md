# Wekraft VS Code Extension

**Wekraft** is the ultimate IDE extension for developers who want to stop lagging behind and start mastering their workflow. Bring project management directly into VS Code so you can code, manage, and collaborate without ever switching tabs.

---

## ✨ Features

### 📋 Tasks
Take complete control over your daily engineering work directly from the sidebar. When managing tasks, you have access to:

| Field | Description |
|---|---|
| **Title** | A concise description of the work. Be specific (e.g. "Fix login button alignment"). |
| **Description** | Rich text notes, acceptance criteria, or context. Supports Markdown. |
| **Status** | The current state of the task (e.g. Not Started, In Progress, Reviewing). |
| **Priority** | High, Medium, or Low. |
| **Start & End Date** | The estimated working window for this task. |
| **Assignees** | One or more team members responsible for this task. |
| **Type / Tag** | A custom label with a colour (e.g., `frontend` / `blue`). |
| **Sprint** | Assign to an existing sprint, or leave blank to keep it in the backlog. |
| **Codebase Link** | A file path relative to your repo root. Clickable right inside the VS Code extension. |
| **Attachments** | Upload images, PDFs, or any reference file. |

### 🐛 Issues
Don't let bugs slip through the cracks. Log and update issues natively with context:

| Field | Description |
|---|---|
| **Title** | A concise description of the problem. |
| **Description** | Context, reproduction steps, screenshots. Supports Markdown. |
| **Severity** | Critical (blocking users), Medium (degraded experience), Low (minor). |
| **Environment** | Where the issue occurs: local, dev, staging, or production. |
| **Due Date** | Optional deadline for resolution tracking. |
| **Assignees** | One or more team members responsible for fixing this. |
| **Sprint** | Assign to the current or a planned sprint. |
| **File Linked** | A file path associated with the bug (clickable in VS Code). |

### 🎟️ Tickets
Manage your external backlog or team requests effortlessly.
- View tickets assigned to you or created by you.
- Update ticket statuses (Open/Closed) in real-time.

### ⏱️ Project Deadlines & Sprints
Keep your eyes on the finish line. See the overarching **Project Deadline** counting down at the top of your sidebar, and manage development cycles using **Sprints**:

| Field | Description |
|---|---|
| **Sprint Name** | Must be unique within the project (e.g., "Sprint 3", "Authentication Week"). |
| **Sprint Goal** | A one-sentence statement of what this sprint will achieve. |
| **Start Date** | When work begins. |
| **End Date** | When the sprint closes. Must be after the start date and cannot exceed the project's target date. |

### 👥 Team Members
Collaboration made simple.
- View your entire teamspace in the sidebar.
- See who is currently assigned to the project and check their roles.

## ⚙️ Getting Started

1. Install the extension from the VS Code Marketplace.
2. Click the **Wekraft** icon in the Activity Bar.
3. Click **Login** to authenticate securely via your Wekraft account.
4. Select your Project and start managing your workflow!
